# 🎉 VIP Birthday Gift Shop

A sleek, responsive web application designed to act as an interactive birthday gift registry. Users can browse a curated list of gifts, view details, click through to the actual store, and "reserve" items to prevent duplicate gifting. 

## ✨ Features

* **Interactive Gift Cards:** Dynamic rendering of gift data including images, descriptions, and prices.
* **Reservation System:** Users can enter their first and last name to mark an item as "Reserved".
* **Data Persistence:** Utilizes browser `localStorage` to save reservations, ensuring data remains intact even after a page reload.
* **Reservation Management:** Includes functionality to cancel a reservation and free up the item.
* **Responsive UI:** Built with CSS Grid to ensure seamless viewing across laptops, tablets, and mobile phones.
* **Premium Theme:** Features a custom black and gold "VIP" aesthetic.

## 🛠️ Technologies Used

* **HTML5:** Semantic page structure.
* **CSS3:** Custom styling, Flexbox, and Grid for responsive design.
* **Vanilla JavaScript (ES6):** DOM manipulation, event handling, and data management.

## 🚀 Getting Started

### Prerequisites
To run this project locally, you only need a modern web browser (Chrome, Firefox, Safari, Edge). No backend server or dependencies are required.

### Installation & Execution
1. Clone the repository:
   \`\`\`bash
   git clone https://github.com/yourusername/vip-birthday-shop.git
   \`\`\`
2. Navigate to the project directory:
   \`\`\`bash
   cd vip-birthday-shop
   \`\`\`
3. Open `index.html` in your preferred web browser.
